➡️ Un navigateur ne peut PAS accéder à un fichier situé en dehors du dossier du site, même si tu mets un chemin absolu comme `/Volumes/USB/...`.

C’est une restriction de sécurité du navigateur.

## ✅ Les 3 façons possibles de lire ta vidéo sur clé USB

1. Copier les vidéos dans ton dossier du site (recommandé)

Tu mets tes .MP4 dans ton projet :

```bash
/mon-site/
   index.html
   /media/
       vidéo1.mp4
       vidéo2.mp4
```

Et ton HTML fonctionne immédiatement.

👉 Cette méthode est la seule vraiment simple et fiable.

## 2. Faire un serveur web qui pointe vers la clé USB

Tu peux lancer un mini serveur local qui permet au navigateur d'y accéder :

```bash
cd "/Volumes/NO NAME/"
python3 -m http.server 8080
```

Puis dans ton site :

```html
src="http://localhost:8080/media/DJI_20241225180443_0083_D.MP4"
```

👉 Ça marche uniquement si la clé est branchée ET le serveur lancé.

## 3) Faire une copie virtuelle de ta clé dans ton projet

Tu peux faire un lien symbolique :

```bash
ln -s "/Volumes/NOM_CLE/media" "/chemin/vers/ton/projet/media_usb"
```

puis utiliser :

```html
src="media_usb/DJI_20241225180443_0083_D.MP4"
```

## 📌 Si tu n’as pas de miniature

Je peux te générer un script Python (ffmpeg) qui extrait automatiquement une image de la vidéo pour faire la vignette.

Exemple :

```bash
ffmpeg -i ../Anniv-Steph/media/DJI_20241225180443_0083_D.MP4 -ss 00:00:00 -vframes 1 ../Anniv-Steph/media/DJI_20241225180443_0083_D-thumbnail.jpg

```

```bash
cd "/Volumes/NO NAME/media_portfolio_photo_video/"
ffmpeg -i Mariage_1.mp4 -ss 00:00:00 -vframes 1 Mariage_1_-thumbnail.jpg

```

```bash
cd "/Volumes/NO NAME/media_portfolio_photo_video/"

for f in *.(mp4|MP4|mov|MOV); do
    [ -f "$f" ] || continue
    base="${f%.*}"
    echo "➡️ Génération thumbnail pour $f"
    ffmpeg -i "$f" -ss 00:00:01 -vframes 1 "${base}_thumbnail.jpg"
done
```

- `Mariage_1.mp4` → `Mariage_1_thumbnail.jpg`
- `DJI_20240101_001.MOV` → `DJI_20240101_001_thumbnail.jpg`
- etc.

## 🧠 Option : choisir une frame plus intéressante

Tu peux changer la frame extraite, par exemple à 2 secondes :

```bash
ffmpeg -i "$f" -ss 2 -vframes 1 "${base}_thumbnail.jpg"
```

#### ⚡️ Option : qualité plus élevée

```bash
ffmpeg -i "$f" -ss 2 -vframes 1 -q:v 1 "${base}_thumbnail.jpg"
```
